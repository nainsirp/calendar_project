# 📅 Calendarr

An Interactive, Premium Wall Calendar Application built with **React** and **Vite**. Calendarr provides an aesthetically pleasing interface with rich seasonal hero images for each month, beautiful hover animations, and a built-in month-by-month "Tasks & Notes" list that automatically saves to your local storage.

## ✨ Features

- **Dynamic Wall Calendar**: Smoothly navigate through months and years.
- **Monthly Hero Images**: Gorgeous, high-quality images change based on the current month context to keep the UI fresh.
- **Tasks & Notes Integration**: Add, check off, and delete monthly tasks seamlessly. Your tasks are stored locally per month, making it a perfect personal organizer.
- **Premium Aesthetics**: Carefully crafted UI using a modern vibrant purple theme, soft drop shadows, hover transitions, and a clean card-based layout.
- **Responsive Design**: Looks beautiful whether running on a desktop screen or a mobile device.

## 🛠️ Technology Stack

- **React 19**
- **Vite** (Build Tool)
- **Vanilla CSS** for beautifully crafted styling and animations

## 🚀 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/YourUsername/calendarr.git
   cd calendarr
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the local development server:
```bash
npm run dev
```
The app will typically be available at `http://localhost:5173/` (or whichever local port Vite opens).

### Building for Production

To generate a production-ready build:
```bash
npm run build
```
The optimized files will be outputted to the `dist` folder, ready to be deployed.
